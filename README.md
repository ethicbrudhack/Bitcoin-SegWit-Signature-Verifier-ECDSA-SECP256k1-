# Bitcoin-SegWit-Signature-Verifier-ECDSA-SECP256k1-
> ⚙️ **Advanced cryptographic tool for validating SegWit Bitcoin transaction signatures**   > This script demonstrates how to **rebuild SegWit transaction preimages**,   > **parse ECDSA signatures**, **decompress public keys**,   > and verify them directly against the **SECP256k1** elliptic curve.
